This file records the physical time and root-level steps associated with
each data dump.

Example:
``` markdown
DumpID                          Time                 Step
     0          0.00000000000000e+00                    0
     1          5.00000000000000e-01                   10
     2          1.00000000000000e+00                   20
```

Table format:
* `DumpID`: data dump ID
* `Time`: physical time
* `Step`: number of root-level updates
